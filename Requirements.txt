 Requirements

This project requires the following libraries:

- **NumPy (1.26.4)** → Numerical computations  
- **Pandas (2.2.2)** → Data manipulation  
- **Matplotlib (3.9.2)** → Visualization  
- **Seaborn (0.13.2)** → Statistical plotting  
- **SciPy (1.14.1)** → Advanced statistical functions  
- **Jupyter Notebook (7.2.2)** & **IPyKernel (6.29.5)** → For interactive analysis  

Additionally, the project includes a **custom module**:  
- `OOPs_deep_stats.py` → Implements OOP-based statistical methods (mean, median, variance, skewness, kurtosis, confidence intervals, visualizations, etc.).
